# FishFM
A Cross Platform Music Discovery FM

> 『鱼声FM』遇见属于你的声音

《鱼声FM》是一款以发现音乐为核心的跨平台音乐软件，音乐类型主要偏纯音乐、后摇滚、电子乐、国风音乐等，如果你喜欢主要面向喜欢「鱼声音乐精选」公众号推荐的音乐，那么这个软件会让你发现更多宝藏音乐。


目前仅支持PC端，支持Win、Linux、Mac，希望体验预览版的可以文末扫码入群。
